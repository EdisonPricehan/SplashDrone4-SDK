# SplashDrone4-ros

## Dependencies
- [ZeroMQ](https://github.com/zeromq/libzmq)
Make sure download the ZeroMQ core engine in C++. Can download the binary package from this [link](https://software.opensuse.org/download.html?project=network%3Amessaging%3Azeromq%3Arelease-stable&package=libzmq3-dev) based on your OS and CPU architecture.
  
- [Cryptodome]()
  
  `pip3 install pycryptodomex`
  
- [gnupg]()
  
  `pip3 install gnupg`
