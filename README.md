# SplashDrone4-ros
This is the ROS1/2 package for the SplashDrone4, that allows you to monitor the teletry data like flying status and camera streams from the drone, while sending control commands to it.

We support both ROS1 on Ubuntu 20.04 and ROS2 Humble on Ubuntu 22.04. (It worked on Windows 11 several years ago but I have not tested it recently)

For specific usage instructions please switch to the corresponding branch.
